# Blackout Vaults

## Getting Started

1. Install dependencies:
   npm install

2. Run development server:
   npm run dev

3. Visit:
   http://localhost:3000

## Environment Variables

Use this `.env.local`:
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_...
STRIPE_SECRET_KEY=sk_live_...
DOMAIN=https://blackoutvaults.com
